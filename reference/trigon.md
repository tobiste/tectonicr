# Trigonometric Functions in Degrees

Trigonometric functions expecting input in degrees.

## Usage

``` r
sind(x)

cosd(x)

tand(x)

asind(x)

acosd(x)

atand(x)

atan2d(x1, x2)

cot(x)

cotd(x)
```

## Arguments

- x, x1, x2:

  Numeric or complex vectors.

## Value

scalar or vector of numeric values.
