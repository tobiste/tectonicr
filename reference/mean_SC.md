# Mean Cosine and Sine

Mean Cosine and Sine

## Usage

``` r
mean_SC(x, w = NULL, na.rm = TRUE)
```

## Arguments

- x:

  angles in degrees

- w:

  weightings

- na.rm:

  logical

## Value

named two element vector

## Examples

``` r
if (FALSE) { # \dontrun{
set.seed(1)
x <- rvm(100, 0, 5)
mean_SC(x)
} # }
```
