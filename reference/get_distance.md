# Helper function to Distance from plate boundary

Helper function to Distance from plate boundary

## Usage

``` r
get_distance(lon, lat, pb.coords, tangential, km)
```

## Arguments

- lon, lat:

  numeric vectors

- pb.coords:

  matrix

- tangential, km:

  logical
