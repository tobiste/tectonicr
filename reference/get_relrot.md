# Helper function to Equivalent rotation

Helper function to Equivalent rotation

## Usage

``` r
get_relrot(plate.rot, lat, lon, angle, fixed, fixed.ep)
```

## Arguments

- plate.rot, fixed:

  character or numeric

- lat, lon, angle:

  numeric

- fixed.ep:

  data.frame
