# Helper function to get Distance from plate boundary

Helper function to get Distance from plate boundary

## Usage

``` r
get_projected_pb_strike(lon, lat, pb.coords, pb.bearing, tangential)
```

## Arguments

- lon, lat, pb.bearing:

  numeric vectors

- pb.coords:

  matrix

- tangential:

  logical
