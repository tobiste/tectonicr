# Plate Boundaries on the Earth

Global set of present plate boundaries on the Earth based on PB2002
model by Bird (2003). Contains the plate boundary displacement types
such as inward, outward, or tangentially displacement.

## Usage

``` r
data('plates')
```

## Format

An object of class `sf`

## References

Bird, P. (2003), An updated digital model of plate boundaries, *Geochem.
Geophys. Geosyst.*, **4**, 1027, doi: 10.1029/2001GC000252, 3.

## Examples

``` r
data("plates")
head("plates")
#> [1] "plates"
```
