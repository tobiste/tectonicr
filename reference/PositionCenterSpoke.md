# Centrically aligned geom_spoke marker

`"position"` subclass "center_spoke" to center `ggplot::geom_spoke()`
marker at its origin

## Usage

``` r
PositionCenterSpoke
```

## Format

An object of class `PositionCenterSpoke` (inherits from `Position`,
`ggproto`, `gg`) of length 2.
